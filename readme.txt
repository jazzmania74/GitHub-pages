나눔경영컨설팅 웹앱 포털
=======================

배포 URL: https://jazzmania74.github.io/GitHub-pages/index.html

나눔경영컨설팅에서 개발한 AI 웹앱 도구들의 링크를 모아 놓은 포털 페이지입니다.

주요 기능:
- 전체 웹앱 카드형 목록 (클릭 시 새 창에서 열림)
- 카테고리별 필터링 (B2B AI 도구 / VWE 계산기 / 고객사 / 개인 도구)
- 통계 표시 (전체 앱 수, AI 도구 수, 카테고리 수)
- 반응형 디자인

업데이트 방법:
- index.html 상단의 APPS 배열에서 앱 추가/삭제/수정
- LAST_UPDATED 날짜 업데이트
- git add → git commit → git push

기술:
- 프론트엔드: HTML/CSS/JS (단일 파일)
- 배포: GitHub Pages

제작: 김종혁 (나눔경영컨설팅)
